# 💱 Currency Converter Using Java

A simple Java console-based application that converts currency values between different currencies.

---

## 📌 Features

* Convert between **USD, EUR, INR, GBP**
* User-friendly console input
* Real-time conversion using predefined exchange rates
* Handles invalid inputs

---

## ⚙️ Technologies Used

* Java
* Scanner (for user input)

---

## 📥 How to Run

1. Clone the repository or download the file

2. Compile the program

```bash
javac Currency.java
```

3. Run the program

```bash
java Currency
```

---

## 🧾 Example

```
========= Currency Converter =========
Available currencies: USD, EUR, INR, GBP

Enter source currency: USD
Enter target currency: INR
Enter amount: 10

10.00 USD = 827.50 INR
```

---

## 📊 Supported Currencies

* USD (US Dollar)
* EUR (Euro)
* INR (Indian Rupee)
* GBP (British Pound)

---

## 🚀 Note

* Exchange rates are **fixed** in the code
* Can be updated for real-time rates in future
